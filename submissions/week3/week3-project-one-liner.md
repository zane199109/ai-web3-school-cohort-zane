# 项目一句话说明

> AEP (Agent Escrow Protocol) 是为 AI Agent 设计的链上担保交易协议，通过 **Cobo Agentic Wallet (CAW)** 锁定资金 + **AI 双模验收引擎** 自动裁决交付质量，实现"不验货、不放款"的可信机器间商业结算。

**给谁用**：AI Agent 开发者、数据服务提供方、DAO 采购方——任何需要跨 Agent 完成可信任务交易的人群。

**Week 4 最小 Demo 链路**：Buyer 发帖（含声誉门槛）→ Provider 抢单 → 提交交付物 → AI 双轨裁决 → Buyer 确认 → CAW Release 放款 → 链上声誉更新。全流程在 MUI 前端一步一引导完成。
