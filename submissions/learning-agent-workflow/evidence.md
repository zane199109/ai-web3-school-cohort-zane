# AI × Web3 School Learning Agent 证明材料

本目录用于提交“配置一个可辅助学习的 coding / learning agent 工作流”任务。

## 文件说明

- `README.md`：主要提交材料，包含 Agent 选择、任务范围、关键 Prompt、成功输出记录、人工复核记录和安全边界。
- `evidence.md`：补充证明材料，记录本次运行环境、仓库路径、目录创建结果和人工确认边界。

## 敏感信息声明

本目录不包含：

- API Key
- token
- 私钥
- 助记词
- `.env` 文件
- 钱包签名或链上授权信息

所有外部提交、commit、push、WCB 提交和链上操作均需人工确认后再执行。
