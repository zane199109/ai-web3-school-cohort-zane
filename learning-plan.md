# 学习计划

1. **AI 基础** — LLM、Prompt、Context、RAG、Agent、Frameworks、MCP
2. **Web3 基础** — Network、Cryptography、Wallet、Smart Contract、Account Abstraction
3. **AI × Web3 Bridge** — Chain-aware Context、Web3 Tool Use、Agent Workflow、Agent Wallet、Machine Payment
4. **前沿探索** — Agentic Commerce、Wallet/Permission、AI Security、Governance

详细内容请参考 [Handbook](https://aiweb3.school/zh/handbook/)。
