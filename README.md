# AI × Web3 School — Learning Journal

> 个人学习笔记与成果仓库 | Personal learning journal and proof-of-work

## 📖 关于

本仓库是 [AI × Web3 School](https://aiweb3.school/) 的个人学习记录仓库，用于：

- 记录每日学习笔记（`daily/`）
- 跟踪学习计划与进度（`learning-plan.md`）
- 沉淀实验与项目代码（`experiments/`、`hackathon/`）
- 整理对 Handbook 的反馈（`handbook-feedback/`）
- 提交课程作业（`submissions/`）
